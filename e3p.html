This is a page for my Ender 3 Pro 3D printer and all it's accessories and updgrades. This printer is connected via USB to a Rasperry Pi loaded with the operating system Octoprint. This allows me to see multiple things through a web interface and gives me full controll over the machine. (Other than phsyically turning it on or off.)

Ender 3 Pro

Upgrade and Planned

Glass Build Plate - Purchased
Glass build plate replacement that will give smoother finished prints on the bottom and give different adhesion for prints.
CR Touch Bed Leveling Sensor - Purchased
Sensor that sends out a small pressure probe to gauge how far off the bed is from the print nozzle and helps auto correct the print. This makes having to level the build plate a thing of the past. This has been purchased and is awaiting install.
v4.2.7 Mainboard Upgrade - Purchased
This goes with the bed leveling sensor and must be installed before that. The board will allow the sensor to work and increase features like thermal protection.

OctoPi Controller

Octoprint
This is the main operating system of the Rasperry Pi and has a few different plugins installed that I will list.
Wifi Status
Displays the status of the device's Wi-Fi status.
OctoPrint-RTMPStreamer
Allows me to stream my webcam to Twitch/YouTube.
OctoPrint-PrintTimeGenius
Provides better print time estimates.
Google Drive Backup
Uploads a backup automatically to Google Drive.
OctoPrint-FilamentManager
Manages my filament by keeping track of how much has been used.
OctoPrint-Dashboard
A dashbaord that displays a lot of information at a glance.
Bed Leveling Wizard
Helps to level the bed by moving to points for adjustment.
OctoPrint-BedLevelVisualizer
This will be helpful once the CR Touch is installed and can provide a visual mesh.
